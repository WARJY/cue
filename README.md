# cue

目前共11个组件，包括<br>
-Page           页面<br>
-AccountPanle   用户登录<br>
-Banner         轮播banner<br>
-Search         搜索<br>
-Input          输入框（共12种类型）<br>
-Upload         图片上传<br>
-Form           表单<br>
-Menu           按钮组<br>
-List           列表（3种类型）<br>
-SearchPanel    筛选框<br>
-TabPanel       选项框<br>


导入微信开发者工具查看即可

-cue   组件库源码<br>
-demo  示例页面，已在小程序编辑器中写好编译模式，导入后可以直接查看
